Privacy Policy

KindKoda Content Automation Tool  ·  Last updated: March 2026

1. Introduction

This Privacy Policy describes how the KindKoda content automation tool ("the Tool") handles information when it interacts with social media platform APIs. This Tool is used exclusively by KindKoda's operator for automated content publishing — it is not a consumer-facing application or service.

2. Information the Tool Accesses

When connecting to platform APIs (TikTok, Meta/Instagram, Facebook, YouTube), the Tool may access:

· Account identifiers — such as TikTok Open ID, Instagram Business Account ID, or YouTube Channel ID, used to authenticate upload requests.
· Access tokens — OAuth tokens stored locally and used only to perform authorized API actions (video upload, post creation).
· Upload confirmation data — video IDs and status codes returned by the platforms after publishing.

The Tool does not collect, store, or process any personal data belonging to third-party users or viewers of the KindKoda channel.

3. How Information Is Used

Information accessed through platform APIs is used exclusively to:

· Authenticate API requests and verify identity
· Upload video content to authorized social media accounts
· Log upload status locally for troubleshooting purposes

No information is sold, shared with third parties, or used for advertising purposes.

4. Data Storage

API credentials and access tokens are stored locally in environment variable files on the operator's private server. They are never transmitted to any third party outside the official platform API endpoints.

Upload logs (video IDs, timestamps, upload status) are stored locally in JSON log files accessible only to the operator.

5. Third-Party Platforms

The Tool interfaces with the following platforms. Each has its own Privacy Policy governing data handling on their end:

· TikTok: tiktok.com/legal/privacy-policy
· Meta (Instagram & Facebook): facebook.com/privacy/policy
· YouTube / Google: policies.google.com/privacy
· OpenAI: openai.com/policies/privacy-policy
· Pexels: pexels.com/privacy-policy

6. No Public User Data

This Tool does not collect, process, or store any personal information from end-users, viewers, or followers of the KindKoda channel. It is solely a backend automation tool used by the channel operator.

7. Security

API credentials are stored in environment variable files protected by the operating system's access controls. The operator is responsible for maintaining the security of these credentials and revoking them if compromised.

8. Changes to This Policy

We may update this Privacy Policy periodically. The "Last updated" date at the top reflects the most recent revision. Continued use of the Tool after an update constitutes acceptance of the revised policy.
